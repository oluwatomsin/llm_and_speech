# Vocal Summarizer
## Project Overview
Vocal Summarizer is a project aimed at creating a tool that can summarize articles, small books, and other textual content, then generate an audio version of the summary. This project leverages OpenAI for natural language processing and chat-based interactions, and ElevenLabs for generating high-quality, human-like audio.

## Features
Text Summarization: Automatically condenses large texts into concise summaries using advanced NLP techniques.
Audio Generation: Converts text summaries into audio, enabling users to listen to summaries on the go.
Customizable Output: Adjust the length and detail of summaries based on user preferences.
## Technology Stack
OpenAI: Used for generating summaries through GPT-based models.
ElevenLabs: Converts text summaries into realistic speech audio.
